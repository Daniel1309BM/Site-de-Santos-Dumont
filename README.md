# ✈️ Santos Dumont – O Pai da Aviação

Este projeto é um site informativo dedicado à história de **Alberto Santos Dumont**, um dos pioneiros da aviação mundial. O site apresenta sua trajetória, contribuições para o desenvolvimento da aviação e alguns princípios científicos relacionados ao voo.

## 📖 Sobre o Projeto

O objetivo do site é divulgar informações históricas e científicas sobre Santos Dumont de forma visual e interativa. A página reúne conteúdos sobre:

- A história e biografia de Santos Dumont  
- Linha do tempo de suas conquistas  
- Princípios físicos relacionados ao voo  
- Elementos visuais e seções informativas para facilitar o aprendizado

O site foi desenvolvido utilizando tecnologias web modernas para criar uma experiência simples, responsiva e acessível.

## 🛠 Tecnologias Utilizadas

O projeto utiliza as seguintes tecnologias:

- **HTML5** – Estrutura da página  
- **CSS3** – Estilização e identidade visual  
- **JavaScript** – Interatividade e funcionalidades  
- **Bootstrap 5** – Layout responsivo e componentes de interface  
- **Google Fonts** – Tipografia personalizada

- Navbar responsiva com menu de navegação
- Seções organizadas com conteúdo histórico e científico
- Layout adaptável para diferentes tamanhos de tela
- Elementos visuais que reforçam a identidade do tema aeronáutico

## 🚀 Como Executar o Projeto

1. Baixe ou clone este repositório.
2. Extraia os arquivos (se necessário).
3. Abra o arquivo `index.html` em qualquer navegador.

Não é necessário instalar dependências adicionais.
